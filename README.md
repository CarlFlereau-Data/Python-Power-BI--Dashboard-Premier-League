# ⚽ Premier League Team Performance Dashboard
🎯 Objectif du projet

Ce projet propose une analyse comparative des équipes de Premier League sur les 10 dernières saisons.

L’objectif est de :

✅ Situer les équipes actuelles par rapport aux saisons précédentes
✅ Analyser leurs performances face à des adversaires en forme offensivement/défensivement
✅ Combiner modélisation statistique (Python) et visualisation décisionnelle (Power BI)

Principaux axes d’analyse :

Classement & évolution des points

Performances face aux équipes en forme

Volume de buts marqués / encaissés

Comparaisons historiques

📦 Données

Source : FBref
Méthode : Web scraping via Python
Outil : Jupyter Notebook

Données collectées :

✔ Résultats des matchs
✔ Buts marqués / encaissés
✔ Points
✔ Forme récente des adversaires
✔ Indicateurs offensifs / défensifs

📅 Période : 10 saisons de Premier League (2015 à 2025) 

🐍 Partie Data Science / Python

Travaux réalisés :

✅ Web scraping FBref
✅ Nettoyage & structuration des données
✅ Feature engineering

Exemples :

Forme offensive / défensive glissante

Indicateurs d’adversaires en forme

Agrégations saisonnières

✅ Analyse statistique exploratoire
✅ Modélisation / métriques descriptives
